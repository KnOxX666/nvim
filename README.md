nvim install
==

**Checkout**

Activate nvim configuration just by checking out this repo:
    
    cd ~/.config
    mv nvim nvim.old #save your old settings
    git clone https://github.com/knoxx666/nvim
    
**Ubuntu**
    
    sudo snap install nvim 

**RHEL7/8/9**
    
    https://snapcraft.io/install/nvim/rhel
